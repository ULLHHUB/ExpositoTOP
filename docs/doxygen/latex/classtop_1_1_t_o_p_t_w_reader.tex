Lee una instancia del problema \doxylink{classtop_1_1_t_o_p_t_w}{TOPTW} desde un archivo. 

Parsea un archivo de texto con un formato específico para inicializar un objeto \doxylink{classtop_1_1_t_o_p_t_w}{TOPTW}. El formato esperado es el de las instancias de Solomon, adaptado para \doxylink{classtop_1_1_t_o_p_t_w}{TOPTW}. La primera línea define el número de vehículos y clientes. Las siguientes líneas definen los datos de cada cliente (coordenadas, tiempo de servicio, puntuación, ventanas de tiempo).
